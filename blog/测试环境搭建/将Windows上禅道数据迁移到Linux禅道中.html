# 将Windows上的禅道数据备份

找到文件位置，并将两个文件复制出来，打开.sql.php结尾的文件，并修改，删除顶部的PHP文件标志

打开服务器上的禅道地址，打开数据库管理（服务器上安装禅道和设置数据库管理中密码之前已介绍，这里不多做说明

https://yuan123haha.github.io/haha-s-study_blogs/blog/%E6%B5%8B%E8%AF%95%E7%8E%AF%E5%A2%83%E6%90%AD%E5%BB%BA/%E5%9C%A8%E6%9C%8D%E5%8A%A1%E5%99%A8%E4%B8%8A%E6%90%AD%E5%BB%BA%E7%A6%85%E9%81%93.html (opens new window)）

打开，zentao数据库，选择导入

选择刚刚复制的文件.sql.php结尾的文件，点击执行

给出导入成功提示

再次打开Linux禅道网址，发现里面数据已被迁移
