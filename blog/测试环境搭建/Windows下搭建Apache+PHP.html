# Windows下搭建Apache+PHP

# 安装Apache

我用的是：apache2.4+mysql5.7+php

Apache2.4版本下载地址：https://www.apachelounge.com/download/

我是创建了一个新文件夹：D:\TESTING\wamp

将Apache解压文件放到wamp文件夹内

打开D:\TESTING\wamp\Apache24\conf\httpd.conf文件，文件内可以修改默认的端口号，现在是80，我改成88

还有个地方地址需要变(服务器地址)

使用管理员权限打开cmd命令窗，将地址转到Apache的bin目录

之后输入httpd -k install命令安装Apache

安装成功，但是有个小错误

找到D:\TESTING\wamp\Apache24\conf\httpd.conf文件，找到ServerName，增加一个ServerName 为localhost，保存退出

再次输入httpd -k start，检查一下，看还有没有报错

又在报其他的错误：

(OS 10048)通常每个套接字地址(协议/网络地址/端口)只允许使用一次。 : AH00072: mak

e_sock: could not bind to address [::]:88

(OS 10048)通常每个套接字地址(协议/网络地址/端口)只允许使用一次。  : AH00072: mak

e_sock: could not bind to address 0.0.0.0:88

AH00451: no listening sockets available, shutting down

AH00015: Unable to open logs

结过看了任务管理器，httpd服务启动了，关闭就行了

有可能报错：(OS 10048)通常每个套接字地址(协议/网络地址/端口)只允许使用一次。 : AH00072: make_sock: could not bind to address 0.0.0.0:443

修改D:\Softwave\Dp\Apache\conf\extra\httpd-ahssl.conf

所有的443 改成 442 自己用替换工具

再次输入httpd -k start

没有报错了，在浏览器中打开网址：localhost：88

Apache安装完成，一般的静态网页就可以通过Apache发布啦；

如果要外网访问的话

如果只是一般的静态页网站（纯HTML、CSS和JavaScript网页）就可以通过该服务器对外发布了。如果要完成对外发布，只需要额外做以下几件事情：

1.购买一台服务器装上Apache

2.购买一个域名，将Apache的配置文件中的ServerName由localhost改成对应的域名

3.在购买域名的服务商那里绑定所购买的域名指向的你服务器对应的IP地址

4.将网站页面放到对应的Web文件夹（DocumentRoot所指向的文件路径）

# 安装PHP

我的安装的是7.2.19，下载地址：https://windows.php.net/download#php-7.2

下载解压到wamp文件夹中，将名字改为php7

在D:\TESTING\wamp\php7里面，复制一份php.ini-development然后修改成php.ini

在wamp下新建一个web文件夹，在web文件夹下新建一个php文件，写一个示例脚本

再在D:\TESTING\wamp\php7下，运行命令，按住shift键+鼠标右键+选择“在此处打开命令窗口”

输入：php -f D:\TESTING\wamp\web\index.php

PHP安装成功

# Apache加载PHP

找到D:\TESTING\wamp\php7文件下的php7apache2_4.dll文件

在D:\TESTING\wamp\Apache24\conf\httpd.conf文件中使用LoadModule命令加载PHP模块:

LoadModule php7_module "D:\TESTING\wamp\php7\php7apache2_4.dll"

并在文件中指定需要PHP解析的文件类型：

AddType application/x-httpd-php .php .html

还需要加载PHP的配置文件：PHPIniDir "D:\TESTING\wamp\php7\php.ini"

修改Apache加载的默认网页地址：

"F:\test\wamp\web">

DocumentRoot "F:\test\wamp\web"

重启apache服务器，在浏览器中输入localhost:88/index.php，网页打开成功

修改index.php文件，打印PHP信息

再次加载网页，发现时区不正确

打开D:\TESTING\wamp\php7\php.ini文件，修改默认时间

重启Apache，再次加载网页
