# 在服务器上搭建禅道

查看服务器版本信息，是多少位的

getconf LONG_BIT

下载Linux，禅道安装包，内置Apache，PHP，mysql，只需要下载解压缩即可运行禅道。（不管之前服务器上有没有安装Apache，PHP，mysql，只要端口号不冲突就可以）

我的是9.1.2版本

http://dl.cnezsoft.com/zentao/9.1.2/ZenTaoPMS.9.1.2.zbox_64.tar.gz (opens new window)

可以安装一个

拖进去就行

进到你的下载目录里，直接解压安装包到/opt目录下

tar -zxvf ZenTaoPMS.9.8.2.zbox_64.tar.gz -C /opt

修改禅道自带apache、mysql端口说明：为了不占用Server上默认的80、3306端口，这里修改端口号为Apache9000，mysql9001；

修改禅道自带的apache端口： /opt/zbox/zbox -ap 9000

修改禅道自带的mysql端口： /opt/zbox/zbox -mp 9001

设置禅道服务自动启动，以保证每次服务器重启后禅道能够正常运行

vim /etc/rc.local
1

在末尾增加一行

/opt/zbox/zbox start
1

保存退出

由于默认情况下rc.local文件无执行权限，还需要给这个文件一个权限

chmod 777 /etc/rc.local
1

为了防止有防火墙的问题，还可以设置一下允许端口访问：

vi /etc/sysconfig/iptables
1

输入：（放在端口为22的下面）

-A INPUT -m state --state NEW -m tcp -p tcp --dport 9000 -j ACCEPT 
-A INPUT -m state --state NEW -m tcp -p tcp --dport 9001 -j ACCEPT
1
2

保存退出

/etc/init.d/iptables restart #最后重启防火墙使配置生效
1
有时候可能其他电脑还打不开、关闭SELINUX 
vi /etc/selinux/config 
##################################### 
（#SELINUX=enforcing #注释掉） 
（#SELINUXTYPE=targeted #注释掉 ）
SELINUX=disabled #增加 
:wq #保存，关闭 
##################################### 
shutdown -r now #重启系统

进入到auth目录

 cd /opt/zbox/auth/
1

执行 ./adduser.sh

然后刚才的页面输入新设置的用户名和密码，就可以进入了
